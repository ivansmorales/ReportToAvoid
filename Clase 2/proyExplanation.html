Report to Avoid

Desarrollo de una aplicación web que ofrecerá al usuario la posibilidad de reportar cualquier tipo de incidencia o queja que se considere moral y/o éticamente incorrecta, que haya vivido en cualquier momento de su día.

ITESM CCM.

TEST

Revisa el portal del Gobierno de la ciudad de México aquí.

1. Yo como usuario con el rol de ciudadano necesito crear una cuenta para poder identificarme como ciudadano.

2. Yo como usuario necesito un usuario y contraseña o bien autentificarme por medio de Twitter para ingresar a mi cuenta.

3. Yo como usuario con el rol de ciudadano necesito un formulario para generar un reporte de incidencia con los datos de la situación y de la persona que realice dicho incidente para que se guarde como evidencia.

4. Yo como usuario con el rol de ciudadano necesito consultar los incidentes para estar informado sobre problemas cerca de mi.

5. Yo como usuario con el rol de ciudadano necesito consultar los incidentes para poder validar si el reporte es verídico.

6. Yo como usuario con el rol de ciudadano necesito cargar una foto de la persona o placa para agregar una evidencia de mi reporte.

7. Yo como usuario con el rol de ciudadano necesito un sistema de valoración para votar por un incidente y que este permanezca como “más importante”.

8. Yo como usuario con el rol del ciudadano necesito un formulario para editar un reporte que ya ha sido enviado.

9. Yo como administrador del sistema necesito acceso a la base de datos para generar un reporte de los usuarios con mayor número de reincidencias.
